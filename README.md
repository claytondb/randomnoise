# Random Fart Soundboard

💨 A fun web-based soundboard with fart sound effects.

## Features

- Multiple fart sound variations
- Browser-based (no installation needed)
- Keyboard shortcuts for quick access
- Adjustable volume
- Randomize mode

## Usage

1. Open `index.html` in a browser
2. Click buttons or use keyboard shortcuts
3. Enjoy the symphony

## Files

- `index.html` - Main interface
- `app.js` - Sound playback logic
- `sounds.js` - Sound definitions and audio data
- `style.css` - Visual styling

## Tech

- Vanilla JavaScript
- Web Audio API
- No external dependencies

## Use Cases

- Pranks (obviously)
- Zoom meeting entertainment
- Party games
- Breaking awkward silences

## Deployment

Static files - deploy to any web host or run locally.
